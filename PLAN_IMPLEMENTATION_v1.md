## Порядок работы в каждой сессии

1. Claude Code подключается к VPS
2. Читает текущее состояние файла
3. Вносит точечные изменения согласно этапу
4. Проверяет сборку (build)
5. Создаёт артефакт (diff, скриншоты, заметки)
6. СТОП — ожидание подтверждения от заказчика
